

## Booking Widget

A free, simple booking calendar widget for Squarespace with Google Calendar integration.

### Features
- 100% free - no monthly subscriptions
- Simple calendar showing only available time slots
- Easy configuration
- Google Calendar sync (optional)
- Email notifications
- Mobile responsive

### Quick Start

See **[README-BOOKING-WIDGET.md](README-BOOKING-WIDGET.md)** for complete setup instructions.

### Files
- `booking-widget.html` - Main widget file
- `booking-widget.css` - Styling
- `booking-widget.js` - Functionality
- `config.js` - Easy configuration
- `example-embedded.html` - Example of embedding in a page# calenderwidget
